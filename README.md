## 👋 안녕하세요!
**React와 JavaScript 기반으로 사용자 중심 인터페이스를 구현하는 프론트엔드 개발자 이겨라입니다.**

실시간 데이터 시각화, UI/UX 설계, 비동기 통신 최적화 프로젝트를 통해  
**사용자가 직관적으로 이해할 수 있는 화면과 흐름을 만드는 데 집중하며 성장하고 있습니다.** FastAPI, Spring 기반 API 연동 경험을 통해 프론트엔드와 서버 간 데이터 흐름 구조를 깊이 있게 이해하며 개발합니다.

**🏆 프로젝트 발표회 최우수상 2회 수상 (성과 검증 완료)**

---

## 🚀 대표 프로젝트 1  
### IoT 기반 실시간 공기질 모니터링 시스템 (팀 프로젝트)

> **React 기반 대시보드로 대용량 IoT 데이터를 실시간 시각화한 프로젝트** > Chart.js 최적화 · 이상치 탐지 알림  
> 🏆 **최종 프로젝트 경진대회 최우수상 수상**

🔗 [프로젝트 저장소](https://github.com/2024-SMHRD-KDT-BigData-29/Smhrd_F4)

### ✅ 담당 역할 (Front-end Lead)
- **React 기반 실시간 모니터링 UI 단독 구축**
- **Chart.js 커스터마이징** 및 이상치(미세먼지/온도 등) 시각적 경고 시스템 구현
- 대량 데이터 유입에 따른 **렌더링 성능 최적화** 및 컴포넌트 설계
- Figma 기반 UI/UX 흐름 설계 및 대시보드 인터페이스 최적화

### 💡 Trouble Shooting & Growth
- **성능 개선:** 지속적으로 유입되는 대량 데이터 처리 시 발생하는 브라우저 부하를 렌더링 주기 최적화로 해결.

## 📱 대표 프로젝트 2  
### 개인 맞춤형 중고 스마트폰 거래 플랫폼 (Plug)

> **사용자 동선 최적화와 API 연동에 집중한 웹 서비스 프론트엔드 프로젝트** > WebSocket 실시간 통신 UI/UX 설계 · REST API 연동 · 반응형 웹  
> 🏆 **핵심 프로젝트 경진대회 최우수상 수상**

🔗 [프로젝트 저장소](https://github.com/2024-SMHRD-KDT-BigData-29/Plug)

### ✅ 담당 역할 (Front-end)
- **HTML5, CSS3, JavaScript 기반 UI 전체 구현**
- **WebSocket 프로토콜**을 활용한 실시간 데이터 스트리밍 구현
- **Spring Boot API 연동**을 통한 상품 목록, 찜, 마이페이지 동적 데이터 처리
- Figma 기반 사용자 흐름(User Flow) 분석 및 인터페이스 설계 주도
- 다양한 디바이스 환경을 고려한 **반응형 웹 구조** 적용

### 💡 Trouble Shooting & Growth
- **UX 개선:** 사용자 피드백을 바탕으로 복잡했던 상품 등록 프로세스를 3단계로 간소화하여 사용자 편의성 증대.
- **실시간성 확보:** HTTP 폴링의 한계를 WebSocket 연동을 통해 극복하고 데이터 실시간 반영 속도 향상.
- **협업 효율화:** 백엔드와의 API 명세 불일치 문제를 Swagger 활용과 긴밀한 소통으로 해결.
